태그의 종류

  1. 강조태그 : strong

  2. 밑줄태그 : u(underline)

  3. 제목 및 줄바꾸기태그 : h1, h2, h3.....(숫자순으로 크기결정)

  4. 단락나누기태그 : p, br(br태그는 여백결정가능)

  5. 목차를 나타내는태그 : ol(자동으로 번호정렬), ul, li(list)

  6. 이미지 삽입태그 : img (예 : img src(source)="사진이름.jpg" width="원하는크기")

  7. 웹페이지를 만든 특성을 알려주는 태그 : meta (예 : meta charset="특성")

  8. 링크를 거는 태그 : a (예 : a href="파일명" title="툴팁표시명" target="_blank" )
